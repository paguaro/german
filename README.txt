German Trainer - Web App
=======================

Contenuto del pacchetto:
- index.html
- app.js
- styles.css
- words.json (2000 coppie tedesco -> italiano)
- README.txt

Istruzioni rapide (su iPhone):
1. Scarica e estrai il pacchetto ZIP sul tuo iPhone (es. tramite Safari).
2. Apri la cartella e tocca `index.html` (si aprirà in Safari). Se il file non si apre localmente, carica la cartella su iCloud Drive e aprila da lì in Safari.
3. Una volta aperta la pagina in Safari, tocca il pulsante Condividi e scegli "Aggiungi a Home" ("Add to Home Screen"). L'app funzionerà offline e salverà i progressi localmente.
4. L'app mostra 20 parole per pagina. Dopo memorizzazione, premi "Passa al Quiz". Per passare alla pagina successiva devi ottenere almeno il 90% di risposte corrette. In caso contrario la stessa pagina verrà riproposta.
5. Per resettare il progresso, usa il bottone "Reset Progresso".

Nota sulla qualità delle traduzioni:
Le traduzioni incluse sono state generate automaticamente con l'obiettivo di coprire 2000 voci. Ti consiglio di rivedere il file `words.json` se desideri correggere traduzioni o varianti (es. accettare forme coniugate o sinonimi). Puoi modificare direttamente il file JSON con un editor di testo o con un'app di gestione file su iPhone.
